# dotfiles

[![Actions Status](https://github.com/nekottyo/dotfiles/workflows/test%20brew%20install/badge.svg)](https://github.com/nekottyo/dotfiles/actions)

install

```bash
$ git clone https://github.com/nekottyo/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ make install
$ make deploy
```
update
```bash
$ make update
```
